# K 线数据下载工具

这是一个从 **Binance 合约 API** 下载 K 线数据的 Python 工具，支持自动合并本地缓存与远程数据，适合量化交易和数据分析。

## 功能
- 下载指定交易对和周期的 K 线数据。
- 本地缓存数据，减少 API 调用。
- 自动清洗无效数据，确保数据质量。
- 进度条显示下载进度。

## 使用
1. 安装依赖：
   ```bash
   pip install -r requirements.txt

2. 运行:
   ```bash
   python binance-download.py
